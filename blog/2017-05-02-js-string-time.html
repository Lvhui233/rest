字符串对象

var str = 'abcdefghigk';
获得6号位置上的字符
var re = str.charAt(6);
获得6号位置上字符的Unicode编码
var re = str.charCodeAt(6);
将Unicode编码转换成原始字符
var re = String.fromCharCode(100);
获得f在字符串中第一次出现的位置
var re = str.indexOf('f');
获得f在字符串中最后一次出现的位置
var re = str.lastIndexOf('f');
查询字符串中有没有gh字符
var re = str.match('gh');
如果有会返回字符串gh

replace替换

replace默认只替换第一个符合条件的内容
var re = str.replace('abc','*****');
逐一查找并替换：
while(str.match('abc')){ str = str.replace('abc','*******'); }

str.substr(start,length);
