事件监听

var box = document.getElementById('box');
低版本ie
box.attachEvent('onclick',function() { })
w3c标准浏览器
box.addEventListener('click',function() { })
用来整合兼容性可以写成
if(document.all){ box.attachEvent('onclick',function(){ }) } else { box.addEventListener('click',function(){ }) }

DOM对象

对html元素的样式(颜色、大小、位置等等)、内容、 属性来进行动态的改变和操作。

document对象的属性

title
返回或设置当前文档的标题

dom抓元素的几种方法

1、通过id抓元素
document.getElementById('box');
2、通过标签名抓元素，抓到的结果是一个数组
document.getElementsByTagName('li');
3、通过class抓元素
document.getElementsByClassName('hd');
4、通过name抓元素
document.getElementsByName('head');

js对元素内容的操作

var title = document.getElementById('title');
获得title的所有内容，包括里面的标签
var con = title.innerHTML;
获得纯文本内容
ie写法：var con = title.innerText;
w3c标准: var con = title.textContent;
短路兼容性写法： var con = title.innerText ||title.textContent

history 对象

属性：length 返回浏览器历史列表中的 URL 数量
方法：
history.back();加载 history 列表中的前一个 URL
history.forward(); 加载 history 列表中的下一个
history.go(number);

页面跳转

location.href = 'http://baidu.com';
location.replace('http://baidu.com'); 用新文档替换当前文档
location.assign('http://baidu.com'); 加载新文档
location.reload();重新加载
assign()加载文档后会在历史记录里面留下记录， 而replace()是指替换当前文档，不会留下记录

对元素属性的操作

元素.属性名的形式只能获得和设置特定的属性
特点的属性包括：ID、CLASS、A标签的href属性、IMG标签的src属性等
获得任意属性：
var re = title.getAttribute('hd');
设置任意属性：
title.setAttribute('hd','aaaa');
