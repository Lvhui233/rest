html5中的表单新增项

邮箱 ：<input type="email" name="" id="">
网址 ：<input type="url" name="" id="">
数字 ：<input type="number"min="3" max='10' value="5" step="2">
滑块 ： <input type="range" name="range" min="1" max="11" step="2">
日期 : <input type="date" />
颜色 : <input type="color" />
datalist : 定义选项列表，用于输入自动匹配 

	<input type="text" list="books" />
		<datalist id="books">
			<option>人民的名义</option>
			<option>孙老师讲前端</option>
			<option>那个女孩真美</option>
			<option>那条小狗真可爱</option>
			<option>大家真好</option>
		</datalist>

autofocus属性：自动获得焦点
<input type="name" autofocus="on">
form属性：提交formh标签以外的表单 

	<form id="form" action="1.php" method="post”>
		<input type="text" name="city">
		<input type="submit"> 
	</form>
	<input type="number" name="age" form="form"/>
multiple属性： 设置multiple后可以设置多个文件
<input type="file" multiple="multiple" />
placeholder属性：设置文本框信息
<input type="text" name="title" placeholder="请输入标题">
required属性：表示内容不能为空
<input type="text" name="title" required title="标题不能为空">

音频播放

audio标签插入音频，controls属性显示控件，autoplay属性自动播放，loop属性循环播放
<audio src="YesterdayOnceMore.mp3" controls="controls" autoplay="autoplay" loop="loop"></audio>

视频播放

视频格式支持：mp4(H.264编码)、webm、ogg 

	<video src="big_buck_bunny.mp4" poster="dog.jpg" controls="controls" id="v"></video>

视频的控制

方法只能用于原生js抓取的对象
var v= document.getElementById('video');
开始播放
v.play();
暂停播放
v.pause();
快进几秒，快退几秒
获得当前位置
var cur = v.currentTime; 计算新位置
var new_time = cur+3;
将新位置设置回去
v.currentTime = new_time;
加速播放，减速播放（倍率）
v.playbackRate = 1.5;
