对象

一切事物皆对象
对象是一堆函数和变量的结合体

字面量的形式创建对象

json格式的数据写法
var iphone7 = { color:'土豪金', price:'8000元', weight:'150g', play:function(){ alert('王者荣耀'); }, chat:function(){ alert('聊妹子'); } };

构造函数的形式创建对象

function phone(){ this.price = '价格'; this.color = '颜色'; this.weight = '重量'; this.play = function(){ alert('能玩游戏'); }; this.chat = function(){ alert('能聊天'); } } var iphone8 = new phone();
自定义参数写法
function phone(p,c,w){ this.price = p; this.color = c; this.weight = w; this.play = function(){ alert('能玩游戏'); }; this.chat = function(){ alert('能聊天'); } } var iphone8 = new phone('5000元','亮黑色','120g');

对象属性和变量的调用

查看属性或方法
alert(iphone8.color);
alert(iphone8['color']);
iphone8.play();
iphone8['play']();

遍历对象

每遍历到一个属性或方法,x代表的是属性名的字符串形式
for (x in iphone7) { document.write(x+'=>'+iphone7[x]+'
'); }

删除对象属性

删除对象里的属性
delete iphone8.price ;
销毁对象
iphone8 = null;

数组对象

push,unshift,pop,shift,splice都属于破坏性操作,会直接改变原数组内容
向数组最后追加元素
shuiguo.push;
向数组最前面追加元素
shuiguo.unshift;
删除数组最后一个元素
shuiguo.pop;
删除数组第一个元素
shuiguo.shift;
splice中第一个参数表示从几号位置开始添加或删除
第二个参数表示删除几个
第三个参数开始,表示要追加的新元素
shuiguo.splice(2,2,'...','...');

slice截取

非破坏性操作
slice截取的时候,包含开始,但是不包含结尾
var re = shuiguo.slice(1,4);
从1号开始,一直截取到最后
var re = shuiguo.slice(1);

数组转换join

非破坏性操作
将数组转换成字符串
shuiguo.join('连接符号');

数组的连接

非破坏性操作
可以将数组连接到一起
var re = shuiguo.concat(shuiguo);

数学对象

取绝对值
var re = Math.abs(-6);
四舍五入，取整
var re = Math.round(2.49);
向上取整
var re = Math.ceil(2.49);
向下取整
var re = Math.floor(2.49);

取随机数

取0到1之间的随机数，无限接近0和1
Math.random();
随机抽取x-y之间(能取到x和y)的随机整数公式：
Math.floor(Math.random()*(y+1-x)+x);
