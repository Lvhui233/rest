函数

函数是一个工厂，是用来帮我们实现某一种功能的

用基本语法声明函数

function hanshu(){ }

用字面量声明函数

var hanshu = function() { }
执行函数
hanshu();

带参数的函数

自定义函数的书写步骤：
1、确定你要实现什么功能
2、将函数声明出来
3、确定需要的参数
4、安排执行流程
例如：
function plus(x,y) { var re = x+y; return re ; }

形参与实参

如果实参数量多于形参，那么只用前几个实参
如果实参数量少于形参数量,那么未被赋值的形参将会变为undefined

变量作用域

局部变量:在函数内部,用var声明的变量
全局变量:不是局部变量的变量就是全局变量

内置顶层函数

将数据转成数字
var data ='123'; var re = Number(data);
转成整数
var re = parseInt('123.5'); var re = parseInt('15px');
浮点数转整数
var re = parseFloat('6.23');
判断数据是否能转成数字
var re = isNaN('sa123');
当需要将字符串转为数字时，可以直接给字符串*1，也可以达到转换的目的
