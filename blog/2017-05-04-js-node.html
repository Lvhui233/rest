获得实际作用在元素身上的样式

ie: var re = box.currentStyle.width ;
w3c : var re = getComputedStyle(box,null).width ;
兼容：var re = box.currentStyle.width || getComputedStyle(box.null).width ;

元素的位置和尺寸

1、获得元素的left值，offsetLeft返回元素相对于父元素的坐标，若父元素没有定位属性，会返回相对于具有定位属性的前辈元素的坐标
var re = box.offsetLeft;
2、获得元素的实际尺寸
var re = box.offsetWidth;
3、获得可见区域的宽度和高度
document.documentElement.clientWidth;
document.documentElement.clientHeight;

滚动条的位置

获取滚动条的位置
var re = box.scrollTop;
设置滚动条位置
box.scrollTop = 1000;

dom模型，节点

var list = document.getElementById('list');
获得所有的子节点的集合
var re = list.childNodes;
找到父级节点的引用
var re = list.parentNode ;
第一个子节点的引用
var re = list.firstChild ;
最后一个子节点的引用
var re = list.lastChild ;
下一个兄弟节点的引用
var re = list.nextSibling ;
上一个兄弟节点的引用
var re = list.previousSibling ;

事件对象

用来记录一些事件发生时的相关的信息的对象
1、只有当事件发生时才产生，只能在函数内部访问
2、处理函数运行结束后自动销毁
document.getElementById("box").onclick = function(e){
获得兼容的事件对象
var ev = window.event || e;
获得鼠标当对于可视区域的距离
alert(ev.clientX);
获得鼠标和事件源之间的距离
var re = ev.offsetX || ev.layerX;
alert(re);
}

事件对象属性

关于鼠标事件

相对于浏览器的位置
clientX 当鼠标事件发生的时候，鼠标相对于浏览器左侧的距离
clientY 当鼠标事件发生的时候，鼠标相对于浏览器Y轴的位置
相对于事件源的位置
IE:
offsetX offsetY
FF:
layerX layerY
使用layerX和layerY时需要给事件源元素添加定位属性

关于键盘的事件

keyCode 获得键盘码
altKey 判断alt键是否被按下，按下是true反之是false布尔值，用在keydown事件里
ctrlKey
shiftKey
