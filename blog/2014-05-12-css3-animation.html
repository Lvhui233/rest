css3动画

transition

transition过渡效果
transition: width 0.3s 0.1s linear;
transition有四个参数分别是，指定应用过渡的属性、过渡的时间、动画开始前的延迟时间，过渡时的速率变化
指定过渡属性可以直接指定属性名，也可以用all给全部元素指定过渡
速率变化有五种方式分别是：ease默认值(等同于贝塞尔曲线)，linear匀速，ease-in加速，ease-out减速，ease-in-out先加速再减速
可以给多个属性指定，中间用逗号分开

animation指定运动规则

animation的参数和transition类似，但是第一个参数是一个指定规则，需要自己定义,最后一个参数代表重复方式 

	animation: move 0.3s 1s linear infinite;
	@keyframes move{
		from{
			width:200px;
		}to{
			width:900px;
		}	
	}
	如果有多段变化，可以用百分比表示
	@keyframes move{
		0%{
			width:200px;
		}50%{
			width:900px;
		}100%{
			width:200px;
		}
	}

transform元素旋转

transform: rotate(30deg);
deg表示度数，默认顺时针旋转
设置旋转基点,基地的位置是指在元素内部的位置
transform-origin:20px 30px;
