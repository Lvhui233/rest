javascript的引用方法

嵌入式调用
<script type="text/javascript"></script>

在超链接位置调用javascript代码
"javascript:alert('我是超链接')"

在重定向位置调用javascript代码
action="javascript:alert('我是表单')"

在时间后面进行调用

调用外部js文件
<script type="text/javascript" src=".../js"></script>

输出工具

  • alert();直接以弹窗的形式显示文本
  • document.write()输出到页面，会以html的语法解析里面的内容
  • prompt('提示文字','默认值')

放置时的注意事项

  • js可以放在html页面的任何位置进行调用，但是它们依然是一个整体，是相互联系并且相互影响的
  • .在调用页面<script></script>标签对当中不能有任何代码， 在js脚本中不能出现<script></script>标签对,但是他们还是 一个整体，是相互联系，相互影响。

注释

  • 行内注释：//注释内容
  • 块注释：/*注释内容*/

js变量

声明变量的几种方法

  • 先声明，后赋值
    var a; a = 'hello';
  • 声明和赋值同时进行
    var a = 'hello';
  • 一次声明多个变量，然后赋值
    var name,age; name = 'Jack'; age = 18 ;
  • 一次声明多个变量，一起赋值
    var name = 'Mary',age = 16,sex = 'female';
