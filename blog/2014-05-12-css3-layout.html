css3里的选择器

大部分选择器和jQuery里抓取元素的选择器类似

选择的是ul里的空li

伪类选择器

ul li:empty{ }
选择的是ul里class不叫a的li标签
ul li:not(.a){ }

文本选择器

选择第一个字符
#box:first-letter
选择第一行
#box:first-line

在选中元素内容前插入内容
#box:before{content:""}
在选中元素内容后插入内容
#box:after{content:""}
可以为插入的元素设置样式，也可以插入空的内容

文本样式

text-overflow属性规定当文本溢出包含元素时发生的事情
text-overflow: clip|ellipsis|string
参数说明：
clip 修剪文本。
ellipsis 显示省略符号来代表被修剪的文本。
string 使用给定的字符串来代表被修剪的文本。

文字强制不换行
white-space: nowrap

文本阴影

向文本添加阴影
text-shadow: h-shadow v-shadow blur color;
从前到后四个参数分别是 水平位置，垂直位置，模糊距离，阴影颜色

盒子阴影

给元素添加阴影
box-shadow: h-shadow v-shadow blur spread color;
从前到后参数分别是 水平位置，垂直位置，模糊距离，阴影尺寸，阴影颜色

最大最小宽高

min-height:10px;
max-height:10px;
min-width:10px;
max-width:10px;

box-sizing怪异模式

属性允许您以特定的方式定义匹配某个区域的特定元素
box-sizing:content-box|border-box;
参数：content-box正常模式，border-box怪异模式： border和padding计算入width之内

兼容性问题

谷歌、safari
-webkit-border-radius: 10px;
火狐
-moz-border-radius: 10px;
opera
-o-border-radius: 10px;
ie
-ms-border-radius: 10px;

文字分栏

-webkit-column-width 栏宽
-webkit-column-count 列数
-webkit-column-gap 列间距
-webkit-column-rule 分隔线

背景图尺寸

background-size: 100% 100%;
保证宽度100%
background-size:contain;
保证高度100%
background-size:cover;

背景渐变色

background:-moz-linear-gradient(left top,green,transparent);
transparent表示透明色
颜色可以设置多个颜色渐变

outline轮廓线

outline（轮廓）是绘制于元素周围的一条线，位于边框边缘的外围，可起到突出元素 的作用， 轮廓线不同于边框，不占用空间尺寸，只有在获得焦点或者激活时呈现。
