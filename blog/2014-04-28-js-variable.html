变量

变量就是装数据的容器

声明变量的几种方法

先声明，后赋值

var name;
name = 'Lucy';

声明的同时赋值

var name = 'Lucy';

同时声明多个变量并赋值

var name = 'Lucy',age = 18,sex = 'female';

变量的重复声明

一般来讲,变量被创建后,任何时候调用都不需要再var重新声明
如果变量被重新声明,但没有被重新赋值,变量值不变

变量的类型

变量里存储的数据由六种类型，分别是数字、字符串、undefined、null、布尔值和对象。
想确定变量的类型，可以用type of 来查看

将字符串中的变量分割

1、将字符串中对应的内容变成变量
2、在变量左右各砍一刀，让左右分别成为一个独立的字符串，然后用加号连接

算数运算符

加 a+b
减 a-b
乘 a*b
除 a/b
取余 a%b
其中除加法外，若由一个是字符串，另一个是数组，js会自动将字符串数字转化为数字类型进行计算。由于加号还有连接字符串的作用，所以不会直接计算。

变量的自增与自减

当++在变量前时，先自增，后运算
当++在变量后时，先运算，后自增
